To download the (re)analysis data, edit and run
the "wget_reanalysis_met_gliston.script" in this 
directory. This script has the capability to process 
era5, merra2, nldas2, and nora10 using Glen's ftp site. 

Additionally, the "wrf_404_download.py" can download WRF 
CONUS 404 data https://rda.ucar.edu/datasets/ds559.0/. Edit 
the script to identify which wateryear you would like to 
download.
